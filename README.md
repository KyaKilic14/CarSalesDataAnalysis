# Car Sales Data Analysis

A beginner Python data analytics project using pandas and matplotlib.

## What this project does
- Loads used car sales data from a CSV file
- Checks the structure of the dataset
- Finds average prices by brand and fuel type
- Counts transmission types
- Measures the relationship between mileage and price
- Creates a simple chart of average price by brand

## Tools used
- Python
- pandas
- matplotlib

## Why I made this
I wanted to build a small project that demonstrates the fundamentals of data analytics using structured data in a business-relevant context. 
This also shows that I am capable of presenting data using python not just Processing and SQL (PHPMYADMIN).

## Paste in Terminal to run
python3 car_analysis.py
